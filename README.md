# projeto-qa-programador

Projeto de testes automatizados com Cypress usado para atividades do curso.

## Descrição

Repositório contendo configurações e testes E2E escritos com Cypress.

## Pré-requisitos

- Node.js (recomendado v14+)
- npm

## Instalação

```bash
npm install
```

## Como rodar

- Abrir a interface do Cypress (modo interativo):

```bash
npx cypress open
```

- Rodar os testes em modo headless (CLI):

```bash
npx cypress run
```

## Estrutura principal

- `cypress.config.js` - configuração do Cypress
- `cypress/e2e/features/` - features de teste (BDD)
- `cypress/steps/` - passos/steps dos testes
- `cypress/fixtures/` - dados de exemplo
